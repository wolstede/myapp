Flagalot has two components:
  • Memory game;
  • Quiz.
Both components use buttons representing flags of the world: all the countries that are full UN member states at the time of app development. These buttons are all the same size, with the same aspect ratio (width to height), whereas national flags have widely varying aspect ratios. This means that some distortion of key features of the flags may occur.

Please note that the buttons for Monaco and Indonesia are identical, even though their national flags have different aspect ratios, but we ensure that they do not occur together in any round of the memory game or quiz.

Some country names used have had to be shortened, either through the use of abbreviations, e.g. 'C African Rep' for 'Central African Republic', through omission of less distinctive parts of the name, e.g. 'Micronesia' for Federated States of Micronesia', and, in all cases, using only the first name if there are two names separated by 'and', e.g. 'Trinidad' for 'Trinidad and Tobago'.

Memory game

The memory game is a simple multi-level game, but one that is difficult to complete at all levels. In each round, at any level, a start card is shown displaying a certain number of flags, each with its associated country shown underneath. After that, match cards are shown, one at a time, each displaying the same number of flags. When one of these is shown, the objective is to tap on the flag that was also displayed on the previous card (there is always exactly one such flag) before the countdown ends. The result is shown after the countdown ends, either 'Right', 'Wrong' or 'No selection made'. The background of the name of the correct country is shown in green, while the background of the name of the country is shown in red if wrongly selected. The score for the round so far is shown underneath the flags, on the left, while the current high score for this level is shown to the right. If the current score exceeds the previous high score, the high score is also increased. Don't forget to continue trying to remember the flags being displayed while the results are being shown, ready for the next match card.

When selecting the common flag, if you change your mind you may tap on another one, assuming time permits. The currently selected flag has its country name displaying a silver background.

Levels

The memory game has 12 levels: 4:1, 4:2, 5:1, 5:2 and so on through to 9:1 and 9:2. The first number represents the number of flags that are displayed on the card at this level. The second number is an indicator of countdown time: 1 means 5 seconds, while 2 means 4 seconds, so less time is available at the higher level.

At the end of a round, the score for the round and the current high score are displayed. The Play button may be tapped to begin another round at the current level, or, if the round was completed with all answers correct, at the next level up (if any).

Quiz

The quiz is concerned with associating flags and countries. There are quiz cards. Each one asks a question, together with three possible answers. You must tap on the correct answer. There are two types of question, which alternate: one shows a flag and asks you to choose the country whose flag this is (by tapping on buttons labelled 1, 2 or 3 above the names); the other shows a name and asks you to select the correct flag by tapping on one of the three flags displayed. As with the memory game, there is a countdown that limits your time for selecting the right answer. Before the countdown ends, you may change your mind by selecting a different answer. The selected answer is again shown with a silver background.

The result of each quiz card is shown after the countdown ends, with three flags and correct names shown at the bottom. Green and red backgrounds again indicate right and wrong answers. The current score and high score are also displayed.

New game

Tapping the New button displays options for both the memory game and the quiz.

Tapping the Play button starts a new round of the memory game at the current level with the current high score. The remaining memory game buttons start a round at the specified level with zero high score.

Tapping the Quiz button starts a quiz with the current quiz high score. The Quiz:0 button starts a quiz with the high score set to zero.

Game memory

When you exit the app, your current level and high scores are remembered for when you next use the app. However, you cannot interrupt a round some way through and pick it up again where you left off when you start the app again.
