Word Ladder is a puzzle in which there 6 rows, each able to take a 4-letter word, where the words at the top and the bottom are presented, but the words in the middle four rows are hidden. To solve the puzzle, you need to find 4 intermediate 4-letter words such that each word differs from the one above and below by one letter only.

So, given the following word ladder,
SUNK
----
----
----
----
MASH

one solution would be:

SUNK
BUNK
BANK
BASK
BASH
MASH

The solutions are not always unique, so in the above case the solution:

SUNK
SUCK
SUCH
MUCH
MUSH
MASH

would also be acceptable.

In this version, an additional constraint has been added: two consecutive letter changes may not be in the same position. So, a sequence such as:

BUST
GUST
MUST

where the first letter is changed in both cases, would not be acceptable, as you could go straight from BUST to MUST.

You will find that you will have to introduce an additional letter into one, and only one, of the four positions that does not appear in this position in the top or bottom words. So, in the first example solution above, the letter B is introduced into the first position; in the second example solution, the letter C is introduced into the third position.

The system checks, using its dictionaries, that any puzzle generated cannot be solved with only three rows rather than four. So, the following should not be presented as a puzzle:

BUST
----
----
----
----
MATE

since it is possible to go from BUST to MATE with only three intermediate rows (and with no need for an additional letter to be introduced):

BUST
MUST
MAST
MATT
MATE

Playing

Tapping the New button starts a new Word Ladder. The system generates this from its main dictionary (see later). You are presented with the Word Ladder, with four missing words.

The top four rows each have a down arrow on the right. Tapping one of these simply copies the characters in this row down into the row below, ready for you to change one. Likewise, the bottom four rows have up arrows on the left. Tapping one of these simply copies the characters in this row up into the row above, ready for you to change one. These buttons help you to solve the puzzle top-down or bottom-up, requiring that you change only one letter in the row.

If your device has a touch screen, you may also copy a character from a single position to the same position in the row above or below by making a slide gesture from the character being copied in the appropriate direction (i.e. slide up or down, as if trying to move it).

Near the bottom of the screen is a character keyboard. Tapping one of these keys makes this character the one to be inserted; it is initially shown just above the middle of the keyboard so that you can be sure which one has been selected. Once a character has been selected, you can tap any position in the middle four rows to change the character displayed there to the selected one. The blank key on the keyboard allows you to clear a position.

If you need some assistance, you may tap the Hint button. This displays the additional letter, above the keyboard, that must be introduced into the Word Ladder, as discussed above, but the position is not given.

Checking or finding a solution

At any time, you may tap the Check button. When checking your solution, however incomplete it may be, the system displays your solution and carries out the following procedure:

  • First, it checks that your solution is complete (no blank letters). If not, it informs you of this (text on a red background) and does no further checking.
  • Next, it checks line by line to see whether the letter changes meet the required criteria, namely one change per line only. If it finds this constraint to be broken, it informs you of this (text on a red background) and does no further checking.
  • Next, it checks to ensure that the letter change is not at the same position in two consecutive changes. If it finds that this has occurred, it informs you of this (text on a red background) and does no further checking.
  • Finally, if all these checks are satisfied, it checks to see whether the words are valid. If it finds any that are not in the built-in dictionaries, it informs you of these on a yellow background, indicating that the solution may or may not be correct, since although the dictionaries contain over 2500 English words, there may be some words that you count as correct that are not in it. If all words are found in the internal dictionaries, you are informed of this on a green background, together with a statement about whether your solution is the same as the built-in one or different.

You then have a number of options concerning the system solution:

  • You may see the complete system solution by tapping the Ours button below the Word Ladder.
  • You may see any row of the system solution by tapping the arrow to the right or left of the row. You might want to do this, or the next option, if you're stuck, and want some help. You may wish to tap the Clear button to clear the middle 4 rows before doing one of these two options.
  • You may see the letter of the system solution at any position by tapping the position.

Tap the Back button to return to the main puzzle screen, if you wish to continue solving the puzzle.

Tap the New button at any time to load a new Word Ladder puzzle.

Dictionaries

The app has two separate dictionaries of 4-letter words: a main one and a subsidiary one. The subsidiary one includes words that are very rare, vulgar, or otherwise contentious. The system uses only the main dictionary for generating its Word Ladder. Both dictionaries are used when checking whether your solution contains valid words.

Exiting the app

The program state is remembered when you exit the app, so that you can continue where you left off.
