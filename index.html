If you're making a daily count of calorie intake, activity, steps or distance walked, money spent or just about anything else, then Daily Counter is for you.

Daily Counter allows you to keep track of four separate counters, each with its own tab at the top (initially labelled Count A through to Count D).

For each counter, you specify a name, a description, the units of measurement (e.g. Calories, points, minutes, km or dollars), and a target or average number of these units that you're trying to achieve each day. You also need to specify whether you are in general trying to minimize the total (e.g. for Calories when weight-watching, or money expenditure) or maximize it (e.g. for activity, steps or distance walked). You can also choose whether you wish to allow a carry-over, so that if your count is over or under your target on some days, you can compensate for this on other days. In other words, you would allow a carry-over if you're more concerned with the longer-term average than with achieving the daily target.

Once you start the counter, you can associate any number of individual counts for a day, and let the Daily Counter calculate the total count for that day. You can also record a brief comment to remind yourself what a count was for. The time at which you enter the count is also recorded to help you remember the context.

You have a complete week open at any time, which means you can enter counts not only for today, but also for any of the 6 previous days (as long as they are after the start of the count).

Daily Counter shows you the counts for the 7 days on a coloured background (green if you've achieved your daily target or red if you haven't). To the right of the daily total is shown your overall surplus or shortfall at that time compared to your target. If you've allowed a carry-over this will, of course, be compared with your expected total over the complete period; in contrast, if no carry-over is allowed, this will be simply the difference between the daily count and the daily target. Again, the red or green coloured background indicates whether or not you are achieving your target on or by that date. You can also choose to see the total count from the time the counter was started, the average over this time, and the minimum and maximum counts entered for any day.

The last four counts entered are available for you to see (together with the associated time of entry and any associated comment). You may choose to reverse any one of these if you've made a mistake (this counts as an extra count entered), or, alternatively, you can choose to enter a negative count if you only want to remove some of the count entered.

Each count may be reset at any time, which means that you can reset totals, averages etc. to zero, ready to start the count again. This can be useful if you also make an associated count over a longer period, and this period has come to an end (e.g. you might be counting weight-watching points on a daily basis, but also allow a certain number of additional points over the full week). You might also use the reset feature when you wish to change your target, or when you have deviated so far from your intended average that you wish to forget about the past and start again. Alternatively, you might use the reset feature to replace this counter with a completely different one.

Defining and starting a counter

To define a particular counter, tap the coloured tab at the top. On this screen, you must enter a short name (which will also be shown on the tab), a description, the units and the target. Simply tap in the relevant field, and this will open up the appropriate text or number edit screen.

Text editing

The text edit screen gives a brief description of what you are editing (together with the number of characters allowed), and shows the current contents of the field (if any). Enter the text using the keyboard at the bottom. Use the red < button to delete the last character, or the red Clear button to clear the text completely.

Use the a-A-1 button to change the keyboard from lower-case, to upper-case, to numbers and symbols, and back again to lower-case. This will have a yellow background if lower-case letters are shown, a blue background for upper-case letters, and a green background for numbers and symbols. Note: If a field is initially empty, upper-case letters will be displayed when the text edit screen is first opened, and automatically revert to lower-case letters after one character has been added.

Note: if you try to exceed the maximum number of characters, a red warning note is shown below the editing field.

Once you have finished editing, press the Done button. Alternatively, press the Cancel button to leave the field you are editing unchanged.

Number editing

The number edit screen gives a brief description of what you are editing (together with the number of characters allowed), and shows the current contents of the field (if any). Enter the number using the keyboard at the bottom. Use the red < button to delete the last character, or the red Clear button to clear the text completely. Use the yellow +/- button to change from a positive to a negative number, or vice versa.

Note: if you try to exceed the maximum number of characters, a red warning note is shown below the editing field.

Once you have finished editing, press the Done button. Alternatively, press the Cancel button to leave the field you are editing unchanged.

Note: if the number is considered invalid, a red error message is displayed. In particular, note that a decimal point must have a numeral before and after it.

Once you have entered the short name, description, units and target, which are mandatory, you should select the appropriate Aim (Minimize or Maximize) and Carry-over (Allow or Don't Allow), by tapping on your choice.

Once you are ready to start the counter, press the Start Counter button. This fixes the start date as today (according to your system).

Note: At or just after midnight, the system tries to ensure that the latest date is displayed, by, in effect, automatically pressing the tab for the current counter. If you happen to be in the text or number edit screen at that time, you will come out of this as if you had pressed its Cancel button.

Entering a count

Once a counter has been started, you will be prompted to enter a count, and, if you wish, an associated comment. Note: Near the top of the screen are shown the description, target and units, together with a reminder of whether a carry over is allowed ('day' means no carry-over, while 'ave' means carry-over allowed) and whether you are trying to maximize or minimize the count ('max' or 'min').

Initially, only one day will be displayed in the table near the top, with a yellow background, but once the counter has been running for more than one day, you will see the current day (always at the bottom of the list) highlighted with a yellow background, with the earlier dates, with white backgrounds, above this. The yellow background signifies the entry date, i.e. the date for which you wish to enter a count. If you wish to enter a count for an earlier date, press on that date and it will change to a yellow background. At the same time, the Submit... button at the bottom right will display the date for which the next entry will be made.

Once you have specified the entry date, enter the count and, optionally, a comment by tapping on the appropriate field and using the relevant text or number editor, as previously. Once ready, press the Submit... button. Assuming there are no errors (again displayed in red), the table at the top is updated and the counter is ready for your next entry.

Note: you can enter negative counts. This might be to correct an earlier entry or to reflect some other benefit or reduction (e.g. if counting points for weight loss, you might have 'earned' points through exercise, so you can reflect these by adding a negative points count).

More information and functionality

On the count entry screen, you will see a More button. Pressing this takes you to a different screen.

On this screen, you will see a different table, this time showing you the last 4 entries made for this counter (1 is the earliest, 4 is the latest). Each of these entries shows the entry date (i.e. the date with which the count is associated), the date and time at which the entry was made, the count, and, beneath these, any associated comment. All of these will be shown with a white background, unless the entry date is not within the last 7 days, in which case it will have a silver background to indicate that this entry is closed (and no further changes can be made for that date).

As well as reminding you of the last 4 entries, you are also able to reverse any of the open entries. To do this, press on any one of these, and its background turns yellow. At the same time, the Reverse button near the button, initially with a silver background, changes to show which of the 4 will be reversed if it is pressed.

Pressing the Reverse button adds a new entry for the relevant date, which is -1 times the original number input. The comment for this is 'Reversal'.

Beneath the table is a statistics area, which shows you the total count and number of days over which this has been made, together with the average (up to 3 decimal places) and the minimum and maximum counts for any day.

Also on this screen, you will find the Reset button for the counter. If you press this, the statistics are replaced by two buttons: one to cancel the reset, in case you made a mistake, and one to confirm it. If you press the Confirm Reset button, numbers will be reset to zero for the count and you will be taken to the initial screen where the counter is defined so that you can make any further changes and then start the counter again.

Press the Back button to return to the count entry screen.

Note: When you shut Daily Counter and then open it again, it should open with the last counter you used displayed.
